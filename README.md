Consultas PagSeguro
=======

Serviço gratuíto de consultas de status do PagSeguro.

A consulta pode ser realizada pelo ID da transação, que pode ser obtido no email e também 
pelo ID da Notificação, que é a requisição que o PagSeguro envia ao seu site cada vez que 
o status de uma compra é alterado (desde que esteja configurada tal funcionalidade). Para obter 
o ID de notificação normalmente basta ler o arquivo  "PagSeguro.log" contido na pasta PagSeuroLibrary 
se você utilizou integração via a API do mesmo.


Para realizar consultas acesse:

http://consultapagseguro.andrebian.com/
